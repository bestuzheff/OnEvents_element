## Типы данных

| Тип | Описание |
|---|---|
| Строка | Текст фиксированной или переменной длины |
| ФорматированнаяСтрока | Текст с HTML-форматированием, неограниченная длина |
| Число | Числовое значение |
| Дата | Дата без времени (ГГГГ-ММ-ДД) |
| ХранилищеЗначения | Бинарные данные (файлы, картинки) |
| Перечисление.* | Ссылка на значение перечисления |
| СправочникСсылка.* | Ссылка на элемент справочника |
